## Задание
Постройте сводную таблицу отображающую топ 10 рейсов по коду рейса (TAIL_NUMBER) и числу вылетов за все время. Отсеките значения без указания кода рейса.

Пример вывода:
| TAIL_NUMBER | count |
|-------------|-------|
| N480HA      | 1763  |
| N484HA      | 1654  |
| N481HA      | 1434  |

Сохраните сводную таблицу в формате parquet.
